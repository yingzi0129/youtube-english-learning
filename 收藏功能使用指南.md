# 收藏功能使用指南

## 功能概述

收藏功能允许用户收藏以下三种类型的内容：
1. **视频收藏** - 收藏整个视频
2. **单词/短语收藏** - 收藏字幕中标注的单词和短语
3. **字幕段收藏** - 收藏特定的字幕片段

## 数据库初始化

在 Supabase SQL Editor 中按顺序执行以下文件：

```bash
# 1. 创建收藏表
database/tables/create-favorites-table.sql

# 2. 设置 RLS 策略
database/policies/favorites-rls-policies.sql
```

或者直接执行：
```bash
database/setup-favorites.sql
```

## 我的收藏页面

访问路径：`/favorites`

### 功能特性
- **三个标签页**：视频、单词/短语、字幕段
- **收藏管理**：查看、删除收藏
- **备注功能**：为字幕段添加学习笔记

  - 导出字幕段为 Markdown 学习笔记

## API 接口

### 获取收藏列表
```
GET /api/favorites?type=video|vocabulary|subtitle_segment
```

### 添加收藏
```
POST /api/favorites
Body: {
  type: 'video' | 'vocabulary' | 'subtitle_segment',
  video_id: string,
  vocabulary_id?: string,
  subtitle_id?: string,
  notes?: string
}
```

### 删除收藏
```
DELETE /api/favorites/:id
```

### 更新备注
```
PATCH /api/favorites/:id
Body: { notes: string }
```

### 批量检查收藏状态
```
POST /api/favorites/check
Body: {
  type: 'video' | 'vocabulary' | 'subtitle_segment',
  ids: string[]
}
```

## 前端组件

### FavoriteButton
收藏按钮组件，支持三种尺寸和显示文字。

```tsx
<FavoriteButton
  type="video"
  videoId={videoId}
  itemId={itemId}
  size="md"
  showText={true}
/>
```

### useFavoritesStore
Zustand 状态管理，提供收藏的增删改查功能。

```tsx
const { favorites, addFavorite, removeFavorite, checkFavoriteStatus } = useFavoritesStore();
```

## 导出格式

### Anki CSV 格式
```csv
Front,Back,Phonetic,Example
"word","meaning","phonetic","example sentence"
```

### Markdown 笔记格式
```markdown
# 我的学习笔记

生成时间：2026-03-06

## 视频标题

### 片段 1 (0:30)

**英文：** English subtitle text

**中文：** 中文字幕文本

**笔记：** 用户添加的备注

---
```

## 注意事项

1. 收藏功能需要用户登录
2. 每个用户只能管理自己的收藏
3. 视频删除时，相关收藏会自动删除（级联删除）
4. 收藏状态使用乐观更新，提升用户体验
5. 导出功能在客户端执行，不会上传到服务器
